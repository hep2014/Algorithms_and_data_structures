## Описание  
  
В работе реализована программа на языке C, которая:  
  
1. считывает числа из текстового файла в **кольцевую очередь**, построенную на базе связного списка;  
2. выводит исходную последовательность;  
3. сортирует данные методом **терпеливой сортировки**;  
4. выводит отсортированную последовательность;  
5. измеряет время выполнения сортировки;  
6. записывает количество элементов и затраченное время в файл `result.csv`.  
  
Работа демонстрирует:  
- реализацию кольцевой очереди на связном списке;  
- обработку файлового ввода;  
- проверку корректности числовых данных;  
- применение алгоритма patience sort;  
- базовую фиксацию результатов измерения производительности.  
  
---  
  
## Постановка задачи  
  
Необходимо разработать программу, которая:  
  
- считывает числа из файла;  
- сохраняет их в кольцевую очередь;  
- сортирует значения методом терпеливой сортировки;  
- выводит данные до и после сортировки;  
- измеряет время сортировки;  
- сохраняет результаты измерения в CSV-файл.  
  
---  
  
## Используемые структуры данных  
  
### Кольцевая очередь  
  
Очередь реализована на базе связного списка и хранит указатель `rear` на последний элемент.  
  
Особенности:  
- если очередь пуста, `rear == NULL`;  
- первый элемент очереди доступен как `rear->next`;  
- последний элемент доступен как `rear`.  
  
### Основные операции кольцевой очереди  
  
- `createQueue()` — создание пустой очереди;  
- `isEmpty()` — проверка очереди на пустоту;  
- `enqueue()` — добавление элемента;  
- `dequeue()` — удаление первого элемента;  
- `front()` — получение первого элемента;  
- `back()` — получение последнего элемента;  
- `printQueue()` — вывод содержимого очереди;  
- `freeQueue()` — освобождение памяти.

## Алгоритм сортировки  
  
### Терпеливая сортировка  
  
В программе реализован алгоритм **patience sort**.  
  
Идея алгоритма:  
1. элементы исходной последовательности последовательно раскладываются по стопкам;  
2. для выбора подходящей стопки используется бинарный поиск;  
3. затем минимальные элементы с вершин стопок последовательно извлекаются и формируют отсортированную последовательность.  
  
### Как это реализовано в программе  
  
1. Содержимое кольцевой очереди копируется во временный массив.  
2. Для каждого элемента определяется подходящая стопка.  
3. После разбиения на стопки очередь очищается.  
4. Отсортированные значения заново помещаются в очередь.

## Проверка входных данных  
  
Программа проверяет:  
- существование файла;  
- наличие чисел во входном файле;  
- корректность каждого считанного токена.  
  
Если встречается ошибка, программа завершает работу с сообщением.  
  
---  
  
## Формат входных данных  
  
Файл должен содержать числа типа `double`, разделённые:  
- пробелами;  
- символами табуляции;  
- переводами строк.

## Формат работы программы

### Шаг 1. Запуск

После запуска программа запрашивает имя файла:

```
Введите название файла с числами
```

### Шаг 2. Чтение данных

После чтения программа:
- выводит число успешно считанных элементов;
- показывает последовательность до сортировки.
    

### Шаг 3. Сортировка

Далее выполняется терпеливая сортировка и выводится отсортированная последовательность.

### Шаг 4. Сохранение результатов

После завершения сортировки в файл `result.csv` записываются:
- количество элементов;
- время сортировки.

## Сборка
```
gcc src/main.c -o main
./main
```

## Формат файла `result.csv`

Программа дописывает результаты в файл `result.csv` в формате:
```
количество_элементов, время_сортировки
```
Пример:

```
5, 0.000012  
100, 0.001245  
1000, 0.054321
```

## Обработка ошибок

Программа завершает работу с ошибкой в следующих случаях:
- файл не найден;
- файл пустой;
- файл не содержит чисел;
- в файле встречается токен, который не является числом.

## Сложность алгоритма

Терпеливая сортировка в такой реализации включает:
- бинарный поиск подходящей стопки;
- последовательный поиск минимальной вершины среди стопок.

Итоговая сложность зависит от количества стопок и входных данных
