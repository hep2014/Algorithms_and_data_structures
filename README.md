# Algorithms and Data Structures 

Репозиторий содержит работы по алгоритмам и структурам данных».

## Список работ

- 1 - Patience Sort for static and dynamic arrays
- 2 - Patience Sort using a circular queue based on a linked list
- 3 - Stooge Sort with naive substring search using a linked-list queue
- 4 - Patience Sort using a stack with writing result into a red-black tree

## Структура репозитория

Каждая работа содержит:
- `src/` — исходный код программы
- `tests/` — тестовые входные данные
- `README.md` — краткое описание, сборка, запуск, примеры тестов

## Тесты

Примеры тестовых файлов лежат в папке tests/.
